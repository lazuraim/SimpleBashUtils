# Simple Bash Utils

Development of Bash text utilities: cat, grep.

## Cat utility

- Support of the following flags:  `-b`, `-e`, `-n`, `-s`, `-t`

## Grep utility

- Support of the following flags: `-e`, `-i`, `-v`, `-c`, `-l`, `-n`, `-h`, `-s`, `-f`, `-o`, including their _pair_ combinations (e.g. `-iv`, `-in`)
